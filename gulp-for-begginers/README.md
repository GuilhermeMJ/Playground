
Gulp for Begginers
====================

Simple Gulp tutorial from [CSS Tricks](https://css-tricks.com/gulp-for-beginners/).

Gulp Plugins covered by it
---------------------------

You can simply look at `package.json` file to see devDependencies, but here is a summary:

* browser-sync
* del
* gulp-cache
* gulp-cssnano
* gulp-imagemin
* gulp-sass
* gulp-uglify
* gulp-useref
* run-sequence
